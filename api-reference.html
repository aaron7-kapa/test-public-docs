API Reference

REST endpoints for the Nimbus platform.test5

Base URL

https://api.nimbus.dev/v1

All requests require an Authorization header with your API key:

Authorization: Bearer YOUR_API_KEY

Endpoints

Method Path Description
GET /workflows List all workflows
POST /workflows Create a workflow
GET /workflows/:id Get a workflow by ID
PATCH /workflows/:id Update a workflow
DELETE /workflows/:id Delete a workflow
POST /events Send a trigger event
GET /executions List workflow executions

Example: Create a Workflow

curl -X POST https://api.nimbus.dev/v1/workflows \
  -H "Authorization: Bearer $NIMBUS_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Welcome Email",
    "trigger": "user.signup",
    "steps": [
      { "action": "send_email", "template": "welcome-v2" }
    ]
  }'

Error Codes

Code Meaning
400 Invalid request body
401 Missing or invalid API key
404 Resource not found
429 Rate limit exceeded
500 Internal server error
Tip: Use the X-Request-Id header in support tickets to help us trace issues faster.
