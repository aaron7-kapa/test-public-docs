Getting Started

Everything you need to ship your first Nimbus workflow in under 5 minutes. Changed.

Installation

Install the Nimbus SDK via npm:

// Install the SDK
npm install @nimbus/sdk

Authentication

Grab your API key from the Dashboard → Settings → API Keys page, then initialize the client:

import { Nimbus } from '@nimbus/sdk';

const nimbus = new Nimbus({
  apiKey: process.env.NIMBUS_API_KEY,
});
Tip: Never hard-code your API key. Use environment variables or a secrets manager.

Create a Workflow

Workflows are the core primitive in Nimbus. Each workflow is a directed graph of steps that execute in order.

const wf = await nimbus.workflows.create({
  name: 'Welcome Email',
  trigger: 'user.signup',
  steps: [
    { action: 'send_email', template: 'welcome-v2' },
    { action: 'wait', duration: '2d' },
    { action: 'send_email', template: 'onboarding-tips' },
  ],
});

console.log(wf.id); // wf_k8x3…
