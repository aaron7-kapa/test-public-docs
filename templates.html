Templates

Reusable email and message templates with dynamic variables.

Overview

Templates define the content sent by send_email and send_sms workflow steps. They support dynamic variables so you can personalize each message.

Creating a Template

const tpl = await nimbus.templates.create({
  slug: 'welcome-v2',
  subject: 'Welcome to Nimbus, {{user.name}}!',
  body: 'Hi {{user.name}}, thanks for signing up.',
  channel: 'email',
});

Variable Syntax

Use double curly braces to insert dynamic values. Variables are resolved from the event payload at runtime.

VariableSource
{{user.name}}User's display name
{{user.email}}User's email address
{{event.name}}Name of the trigger event
{{payload.*}}Any field from the event payload

Listing Templates

const templates = await nimbus.templates.list();
// Returns all templates for your account

Channels

Templates support the following channels:

ChannelDescription
emailHTML or plain-text email
smsText message (160 char limit)
pushMobile push notification
Note: Template slugs must be unique within your account. Use versioned slugs like welcome-v2 to manage iterations.
